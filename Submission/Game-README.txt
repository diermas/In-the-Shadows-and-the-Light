Controls:

WASD - Move
Spacebar - Toggle buttons, start/stop pulling movable blocks, use the exit door
R - Reset the level

How to Play:

Find and walk over the key in each level to pick it up, then make your way to the door and unlock it to advance to the next level.
Blocks with handles can be moved by pushing them or toggling Spacebar to pull them.
Only 1 key is needed in each level to unlock the door.
